# Predictive Analytics Dashboard (Power BI)

## Dashboard Preview

![Predictive Analytics Dashboard](dashboard.png)

## Project Overview

This project demonstrates predictive analytics using historical sales data in Power BI. The dashboard analyzes business performance, identifies historical trends, and forecasts future revenue using Power BI's built-in forecasting capabilities.

## Objectives

- Analyze historical sales performance
- Identify revenue and profit trends
- Forecast future revenue using time-series analysis
- Support data-driven business decisions through interactive visualizations

## Dashboard Features

- Total Revenue KPI
- Total Profit KPI
- Total Orders KPI
- Monthly Revenue Trend with 6-Month Forecast
- Revenue vs Profit Analysis
- Interactive Date Slicer

## Dataset

The dataset contains monthly historical business data with the following fields:

- Date
- Revenue
- Sales
- Orders
- Profit
- Marketing Spend

## Tools Used

- Microsoft Power BI
- Microsoft Excel
